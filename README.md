 This project is based on the SDG 2: 'Zero Hunger' and the main purpose is  to connect the restaurants where food waste which was left over to the NGO's where the food will be distributed to the needy and the hungry. 
 Our team created the app and conducted various test cases where it was real time accessible and it had the various interfaces of login page, donor page, restaurant page, organization page according to the need of the user and it was completely functional. The project included various phases of :Planning
 & Requirement Analysis, Design Phase ,Backend and Frontend Development ,Feature Implementation, Testing & Refinements,Deployment Preparation and Launch & Post-Launch Activities. 
 Mainly focused on the database integration and have worked on developing the firestore database collection and updation. Finally as the outcome the 'No Hunger' app was fully functional, 
 with attractive user interfaces, efficient database collection with realtime accessibility.
